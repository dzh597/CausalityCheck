The dataset comprises both the origin data and the data processed by causalityCheck. The source data constitutes a subset of CausalProbe-2024, whilst the data tested in this experiment is the processed data.
Should you wish to generate data compliant with the **CausalityCheck** format using a custom dataset, simply replace the source data with your own dataset and follow these steps:
- Ensure your dataset adheres to the input format required by **CausalityCheck**.
- Process the data using the **CausalityCheck** tool to generate output in the standardised format.
